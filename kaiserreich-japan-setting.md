## 1. 基本情報

| 項目 | 内容 |
|------|------|
| 国名 | 大日本帝国（Empire of Japan） |
| 政体 | 立憲君主制（1936年時点） |
| 元首 | 昭和天皇（1926年即位） |
| 首相 | 犬養毅（1931〜1936年5月暗殺） |
| 与党 | 立憲政友会 |
| 人口 | 約6,500万（内地） |
| 陣営 | 大東亜共栄圏（第四の勢力） |
| 対独関係 | **休戦未締結 — 形式上は交戦状態が継続** |

### 世界大戦後の経緯 【KR】

- 日本は第一次世界大戦（Weltkrieg）で協商側として参戦・戦勝
- しかしドイツ勝利により、膠州湾・太平洋旧独領の返還を余儀なくされる
- 1919年ヴェルサイユ講和会議：英国が日本の領土要求を拒否 → 日本代表団退席
- 1921年**青島協定（Tsingtao Accord）**：日独間の限定的和平。日本は青島・ドイツ太平洋領から撤退
- 休戦条約（Armistice）には署名せず → **形式的にはドイツと交戦状態が継続**

## 2. 政治体制と政党

### 政党一覧

| 政党名 | イデオロギー | 支持基盤 | 財閥との関係 |
|--------|------------|---------|------------|
| **立憲政友会** | 社会保守主義 | 地主・財閥・保守層 | **三井系** |
| **立憲民政党** | 市場自由主義 | 都市リベラル層 | **三菱系** |
| **社会大衆党** | 社会民主主義 | 労働者・農民 | なし |
| **国民同盟** | 権威主義的民主 | 右派ポピュリスト | なし |
| 協和会 | 温情的専制 | 軍部統制派 | 軍需系 |
| 大政翼賛会 | 国家主義 | 維新派（皇道派） | — |

### 政友会 【KR】

- 1936年時点の与党。1926年の「憲政復活」以来3回連続で総選挙勝利
- 総裁：犬養毅（1936年5月15日暗殺） → 鈴木喜三郎が後継
- 三井財閥・地主層が主要支持基盤
- 「政友会は三井グループの延長とみなされていた」

### 民政党 【KR】

- 最大野党。浜口雄幸のもとで改革・近代化路線
- 三菱財閥・帝国海軍と結びつき
- 「民政党は三菱グループ、そして帝国海軍と繋がっていた」

### 社会大衆党 【KR】

- 片山潜が創設（サンディカリスト指導者。暗殺が五・一五事件の引き金）
- 労働者・小作農が支持基盤
- 1940年選挙候補：安部磯雄、1944年選挙候補：片山哲

## 3. 陸軍派閥 — 維新派と統制派

> KRでは **維新派（Ishinha）= 皇道派、中央派（Chuuha）= 統制派** に対応

### 維新派（Ishinha / 皇道派相当）【KR】

| 項目 | 内容 |
|------|------|
| 志向 | 天皇親政の復活・体制打破 |
| イデオロギー | 国家主義（National Populist） |
| 行動様式 | 直接行動・クーデターによる「昭和維新」 |
| 政治的帰結 | 近衛文麿を首相とする大政翼賛会体制 |
| 支持層 | 青年将校・地方出身の中下級士官 |

#### 史実の皇道派主要人物 【史実類推】

| 人物 | 役職・立場 |
|------|----------|
| 荒木貞夫 | 陸軍大将。皇道派の創設者的存在。精神主義を重視 |
| 真崎甚三郎 | 教育総監。荒木の盟友。更迭が相沢事件の引き金 |
| 山下奉文 | 青年将校に影響力。後のマレー作戦指揮官 |
| 小畑敏四郎 | 対ソ戦略の急進派 |

### 中央派（Chuuha / 統制派相当）【KR】

| 項目 | 内容 |
|------|------|
| 志向 | 既存体制との協調・官僚的統制 |
| イデオロギー | 温情的専制（Paternal Autocrat） |
| 行動様式 | 合法的手段 → ただし文民政府が倒れれば軍事クーデター |
| 政治的帰結 | 平沼騏一郎を首相とする協和会体制 |
| 主要人物 | 林銑十郎（陸相）、永田鉄山（1943年に林と交代）※KR世界では1935年の相沢事件が回避された前提 |
| 支持層 | 軍上層部・官僚機構 |

#### 史実の統制派主要人物 【史実類推】

| 人物 | 役職・立場 |
|------|----------|
| 永田鉄山 | 統制派の理論的指導者。総力戦体制を構想。1935年相沢事件で斬殺（史実）。**KR世界では生存前提**（上表参照） |
| 東條英機 | 永田の後継的存在。官僚的軍国主義の体現者 |
| 武藤章 | 軍務局長。統制派の実務家 |

### 両派の根本的違い

| | 維新派（皇道派） | 中央派（統制派） |
|--|---------------|---------------|
| 権力掌握の手段 | 直接行動（クーデター） | 体制内浸透・合法的手段 |
| 国家像 | 天皇親政・精神主義 | 官僚統制・総力戦国家 |
| 対外方針 | 即時拡大・対ソ北進論 | 計画的段階的拡大 |
| 経済観 | 反財閥・農本主義 | 財閥との協調・統制経済 |
| 天皇との関係 | 「天皇のために」（独善的解釈） | 「天皇の意思を制度で実現」 |

## 4. 海軍 — 艦隊派と条約派

> KRでは海軍内部の派閥分裂について詳述がないため、史実をベースに類推する。
> ただしKR世界では**ワシントン海軍軍縮条約が成立していない、または大幅に異なる**ため、
> 対立の文脈が変わることに注意。

### 艦隊派（Fleet Faction）【史実類推】

| 項目 | 内容 |
|------|------|
| 主張 | 軍縮条約反対・戦力比率の対米7割確保 |
| 戦略思想 | **艦隊決戦（Kantai Kessen）** — 大艦巨砲主義 |
| 代表人物 | 加藤寛治（軍令部長）、末次信正、高橋三吉 |
| 軍政上の立場 | 軍令部（作戦・統帥）側 |
| 政治的傾向 | 反英米・強硬的対外姿勢 |

### 条約派（Treaty Faction）【史実類推】

| 項目 | 内容 |
|------|------|
| 主張 | 国際協調・軍備管理の受容 |
| 戦略思想 | 経済力と外交を含めた総合国防 |
| 代表人物 | 加藤友三郎（元首相）、山梨勝之進、堀悌吉 |
| 軍政上の立場 | 海軍省（行政）側 |
| 政治的傾向 | 民政党に近い。三菱との関係 |

### KR世界での海軍の特殊事情 【KR + 史実類推】

| 要因 | 影響 |
|------|------|
| ワシントン条約不成立の可能性 | 艦隊派 vs 条約派の対立軸自体が変質。八八艦隊計画が（一部）実現 |
| 主敵がドイツ | 対米比率ではなく**対独東洋艦隊への優位**が焦点 |
| 英国の崩壊 | 条約派の根拠だった「英米協調」が消滅 → 条約派の立場が弱体化 |
| 海軍急進派の政治行動 | 五・一五事件で海軍将校が犬養首相を暗殺 → 海軍内にも直接行動派がいる |

### 海軍のKR世界における想定される派閥構造 【史実類推】

```
  協調派（旧条約派系）          拡大派（旧艦隊派系）          急進派
  ────────────          ────────────          ────
  ・国際協調維持             ・八八艦隊の完遂            ・五一五事件の実行者
  ・民政党・三菱と連携        ・対独戦備の最優先           ・青年将校的直接行動
  ・漸進的南進論             ・大艦巨砲＋空母の併用        ・陸軍皇道派と共鳴
  ・外交的解決を優先          ・南洋ドイツ領の武力奪取      ・反体制・反財閥

  代表：山梨勝之進的人物      代表：加藤寛治・末次信正的人物  代表：暗殺実行の海軍将校
```

### 艦隊決戦（Kantai Kessen）ドクトリン 【史実類推】

- **基本構想：** 漸減邀撃作戦 — 敵主力艦隊を潜水艦・航空機・水雷戦隊で漸減し、最終的に戦艦群による決戦で殲滅
- **KR世界での適用対象：** 対米ではなく**ドイツ東洋艦隊（シンガポール拠点）**
- **問題点：** ドイツ東洋艦隊は「帝国海軍最強の海外部隊」であり、本国艦隊の増援も想定される

## 5. 海軍戦力と建艦計画

### 主要艦艇 【KR】

| 艦級 | 種別 | 備考 |
|------|------|------|
| 長門型 | 戦艦 | 16インチ砲8門。技術的飛躍を示す |
| 天城型 | 巡洋戦艦 | 高速。東南アジア権益の防衛に柔軟対応 |
| 土佐型 | 戦艦 | 八八艦隊の主力 → 一部は空母に改装 |
| 紀伊型 | 戦艦 | 1921年青島協定後に威嚇目的で起工。紀伊・尾張が1932年就役 |
| 秋型 | 戦艦 | 野心的設計だが最終的に計画中止 |
| 神風型 | 駆逐艦 | — |
| 睦月型 | 駆逐艦 | — |
| 空母群 | 航空母艦 | 土佐型からの改装含む。世界有数の空母運用能力 |

### 八八艦隊計画の変遷 【KR】

1. **Weltkrieg中：** 西洋列強の建艦停滞を突いて海軍力を増強
2. **1921年：** 青島協定。紀伊型を誇示的に起工
3. **1925年：** 英国でのサンディカリスト革命を受け、帝国議会が海軍予算を大幅削減
   - 「人民の力と、圧倒的海軍の必要性の低下」を議会が認識
   - 建造中の戦艦を含む複数艦が計画中止（駿河・近江など）
4. **1932年：** 紀伊・尾張が就役。八八艦隊は部分的達成にとどまる

### 海軍の任務 【KR】

- 太平洋通商路の防衛（膨大な駆逐艦隊を船団護衛に充当）
- ドイツ東洋艦隊（シンガポール拠点）への対抗
- 南洋方面への戦力投射能力の維持

## 6. 財閥と経済構造

### 四大財閥 + 新興財閥

| 財閥 | 主要事業 | 政治的立場 | 軍との関係 |
|------|---------|----------|----------|
| **三井** | 商事・鉱山・銀行 | **政友会（保守）** | **陸軍**と密接 |
| **三菱** | 造船・重工・銀行 | **民政党（リベラル）** | **海軍**と密接 |
| **住友** | 銅山・金属・銀行 | 比較的中立 | 特定軍閥との強い結びつきは薄い |
| **安田** | 銀行・保険 | 比較的中立 | 金融面での軍需支援 |
| **日産** | 自動車・重工業 | — | **満洲**で急成長（新興財閥） |
| **川崎** | 造船・車両 | — | 海軍向け造船 |
| **トヨタ** | 自動車・織機 | — | 新興。軍需転換の可能性 |
| **日立** | 電機・機械 | — | 軍需電機 |

### 三井 — 陸軍 — 政友会 ライン 【KR + 史実類推】

```
三井合名 ──資金──→ 立憲政友会 ──政策──→ 積極財政・大陸進出支持
  │                                          │
  └──軍需調達──→ 帝国陸軍 ←──拡大路線──────┘
```

- 三井は政友会の最大のスポンサー
- 陸軍の大陸進出は三井の商業利権（満洲・中国市場）と一致
- 【史実】1932年、三井合名理事長・団琢磨が右翼テロリストに暗殺（血盟団事件）
  → 財閥と政党の結びつきへの民衆の怒りの表れ

### 三菱 — 海軍 — 民政党 ライン 【KR + 史実類推】

```
三菱財閥 ──資金──→ 立憲民政党 ──政策──→ 国際協調・軍縮・都市発展
  │                                          │
  └──艦艇建造──→ 帝国海軍 ←──海軍力整備────┘
```

- 三菱重工業（長崎造船所等）が帝国海軍の主要な艦艇建造を受注
- 海軍の近代化・技術重視路線と三菱の重工業路線が合致
- 民政党は海軍の協調外交路線を支持

### 住友と安田 【史実類推】

- **住友：** 別子銅山に始まる金属・鉱業中心。住友金属工業は軍需にも関わるが、政治的にはMitsui/Mitsubishiほど特定政党に肩入れせず
- **安田：** 安田財閥は銀行・保険中心の金融コングロマリット。1921年に安田善次郎が暗殺される。政治的な表立った党派色は薄い

### 満鉄（南満洲鉄道株式会社）【KR】

- 大連に本拠を置く**準財閥的国策会社**
- 奉天（Fengtian）領内の近代的製造業と大規模農業をほぼ独占
- 奉天軍内部に**「親日派」vs「親張派」**の対立を生む
- 日本の大陸権益の経済的基盤

### 経済状況（1936年時点）【KR】

| 時期 | 出来事 | 影響 |
|------|--------|------|
| 1918年 | 米騒動 | 経済悪化の兆候 |
| 1919年 | フランス陥落（最大貿易相手国の喪失） | 貿易崩壊・産業不振・大量貧困 |
| 1923年 | 関東大震災 + アナーキスト蜂起 | 壊滅的破壊 |
| 1925年 | 英国革命 + ドイツの経済覇権確立 | さらなる打撃 |
| 1927年 | 震災手形問題 → 台湾銀行破綻 → 銀行恐慌（昭和金融恐慌） | 金融システム動揺 |
| 1931〜36年 | 犬養内閣の介入主義的回復政策 | **緩やかな回復**。ただし農村の困窮は継続 |
| 1936年 | **Black Monday**（ドイツ発の世界恐慌） | 新たな経済危機の波 |

### 経済の構造的弱点

- **資源貧国：** 石油・鉄・ゴム等を海外輸入に依存
- **太平洋補給線の脆弱性：** ドイツ東洋艦隊が脅威
- **農村と都市の格差：** 農村困窮が過激主義の温床
- **貿易先の制約：** 旧協商諸国（英仏）の崩壊で市場喪失

### 大東亜共栄圏 【KR】

- ゲーム開始時点ですでに第四の主要陣営として形成済み
- 軍事・経済・文化の統合体を標榜
- 二面性：**汎アジア主義の理想** vs **日本覇権の道具**
- Weltkriegでの資源不足の教訓から、自給自足経済圏の構築を志向
- 加盟候補：奉天（関係改善時）、沿アムール共和国、その他獲得国

## 8. 対外関係

### ドイツ帝国（ライヒスパクト）— 最大の敵対国 【KR】

| 項目 | 内容 |
|------|------|
| 関係 | **敵対的。形式上は交戦状態が継続** |
| 対立の焦点 | 太平洋権益、中国での影響力競争 |
| ドイツの東洋艦隊 | シンガポール拠点。「帝国海軍最強の海外部隊」 |
| 中国問題 | ドイツが1928年上海会議で清朝復辟を支援 → 日本の大陸権益と正面衝突 |
| 青島協定の不満 | 日本国民に極めて不人気。都市部で抗議・暴動が発生 |

### ロシア — 緊張関係 【KR】

- 日本が北樺太・ウラジオストクを占領中 → ロシアの領土回復要求
- 1928年奉天紛争：日本が奉天を支援しロシアに決定的勝利
- 沿アムール共和国が日本の対ロ緩衝国

### 亡命協商（カナダ・豪州）— 複雑な関係 【KR】

- 旧英仏の亡命勢力とは**反独で利害が一致**する部分あり
- ただし：
  - ヴェルサイユで英国が日本要求を拒否した経緯 → 不信感
  - 豪州は太平洋での日本拡大を警戒
  - 「敵の敵は味方」だが完全信頼には程遠い

### 第三インターナショナル — イデオロギー的対立 【KR】

- サンディカリスム＝日本にとってのイデオロギー的脅威
- 1923年にアナーキスト蜂起を鎮圧した経験
- 片山潜（社会大衆党創設者、サンディカリスト指導者）の暗殺が五・一五事件の引き金
- 国内左派がインターナショナルとの連帯を模索する動きも

### アメリカ — 地域的競合 【KR】

- 太平洋の競合相手（ただしアメリカ自体が内戦前夜で弱体化）
- アメリカの参戦不在がカイザーライヒ世界の根幹

### 奉天政府 — 最も近い同盟国/傀儡 【KR】

- 日本が唯一承認する中国正統政府
- 満鉄が経済を支配 → 奉天軍内に親日派 vs 親張派の対立
- 関係が改善すれば共栄圏加盟、悪化すれば**奉天が対日開戦する可能性**
- 奉天戦争で東北経済はほぼ破産

### 清朝（ドイツ支援）— 敵対的 【KR】

- ドイツ支援による清朝復辟が日本の大陸権益を脅かす
- 1928年上海会議で日本は租界都市での評議会議席を確保

## 10. 1936〜37年の分岐イベント

### 五・一五事件（1936年5月15日）【KR】

- 海軍急進派将校が犬養毅首相を暗殺
- 動機：サンディカリスト指導者・片山潜の死への報復
- 鈴木喜三郎が後継首相に → **国家保安法**を議会に提出

### 国家保安法の採決 【KR】

これが最大の分岐点：

```
              国家保安法
                │
        ┌───────┴───────┐
        可決              否決
        │                │
  民主制維持ルート     政治危機の深刻化
  （政友会 or 民政党     │
   or 社大党の政権）   ┌──┴──┐
                    中央派      維新派
                    クーデター    反クーデター
                    │           │
                  平沼内閣      昭和維新
                 （協和会）    （大政翼賛会）
                               近衛首相
```

### Black Monday（ドイツ発世界恐慌）【KR】

- ドイツ経済の崩壊が全世界に波及
- 日本の脆弱な経済回復を直撃
- 農村困窮の悪化 → 過激主義の燃料
