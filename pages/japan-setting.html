世界大戦後の経緯 【KR】

  • 日本は第一次世界大戦（Weltkrieg）で協商側として参戦・戦勝
  • しかしドイツ勝利により、膠州湾・太平洋旧独領の返還を余儀なくされる
  • 1919年ヴェルサイユ講和会議：英国が日本の領土要求を拒否 → 日本代表団退席
  • 1921年青島協定（Tsingtao Accord）：日独間の限定的和平。日本は青島・ドイツ太平洋領から撤退
  • 休戦条約（Armistice）には署名せず → 形式的にはドイツと交戦状態が継続

政友会 【KR】

  • 1936年時点の与党。1926年の「憲政復活」以来3回連続で総選挙勝利
  • 総裁：犬養毅（1936年5月15日暗殺） → 鈴木喜三郎が後継
  • 三井財閥・地主層が主要支持基盤
  • 「政友会は三井グループの延長とみなされていた」

民政党 【KR】

  • 最大野党。浜口雄幸のもとで改革・近代化路線
  • 三菱財閥・帝国海軍と結びつき
  • 「民政党は三菱グループ、そして帝国海軍と繋がっていた」

社会大衆党 【KR】

  • 片山潜が創設（サンディカリスト指導者。暗殺が五・一五事件の引き金）
  • 労働者・小作農が支持基盤
  • 1940年選挙候補：安部磯雄、1944年選挙候補：片山哲

軍政上の立場 海軍省（行政）側

艦隊決戦（Kantai Kessen）ドクトリン 【史実類推】

  • 基本構想： 漸減邀撃作戦 — 敵主力艦隊を潜水艦・航空機・水雷戦隊で漸減し、最終的に戦艦群による決戦で殲滅
  • KR世界での適用対象： 対米ではなくドイツ東洋艦隊（シンガポール拠点）
  • 問題点： ドイツ東洋艦隊は「帝国海軍最強の海外部隊」であり、本国艦隊の増援も想定される

艦級 種別 備考

土佐型 戦艦 八八艦隊の主力 → 一部は空母に改装

八八艦隊計画の変遷 【KR】

  1. Weltkrieg中： 西洋列強の建艦停滞を突いて海軍力を増強
  2. 1921年： 青島協定。紀伊型を誇示的に起工
  3. 1925年： 英国でのサンディカリスト革命を受け、帝国議会が海軍予算を大幅削減
    • 「人民の力と、圧倒的海軍の必要性の低下」を議会が認識
    • 建造中の戦艦を含む複数艦が計画中止（駿河・近江など）
  4. 1932年： 紀伊・尾張が就役。八八艦隊は部分的達成にとどまる

海軍の任務 【KR】

  • 太平洋通商路の防衛（膨大な駆逐艦隊を船団護衛に充当）
  • ドイツ東洋艦隊（シンガポール拠点）への対抗
  • 南洋方面への戦力投射能力の維持

財閥 主要事業 政治的立場 軍との関係

三井 — 陸軍 — 政友会 ライン 【KR + 史実類推】

三井合名 ──資金──→ 立憲政友会 ──政策──→ 積極財政・大陸進出支持
  │                                          │
  └──軍需調達──→ 帝国陸軍 ←──拡大路線──────┘
  • 三井は政友会の最大のスポンサー
  • 陸軍の大陸進出は三井の商業利権（満洲・中国市場）と一致
  • 【史実】1932年、三井合名理事長・団琢磨が右翼テロリストに暗殺（血盟団事件） → 財閥と政党の結びつきへの民衆の怒りの表れ

三菱 — 海軍 — 民政党 ライン 【KR + 史実類推】

三菱財閥 ──資金──→ 立憲民政党 ──政策──→ 国際協調・軍縮・都市発展
  │                                          │
  └──艦艇建造──→ 帝国海軍 ←──海軍力整備────┘
  • 三菱重工業（長崎造船所等）が帝国海軍の主要な艦艇建造を受注
  • 海軍の近代化・技術重視路線と三菱の重工業路線が合致
  • 民政党は海軍の協調外交路線を支持

住友と安田 【史実類推】

  • 住友： 別子銅山に始まる金属・鉱業中心。住友金属工業は軍需にも関わるが、政治的にはMitsui/Mitsubishiほど特定政党に肩入れせず
  • 安田： 安田財閥は銀行・保険中心の金融コングロマリット。1921年に安田善次郎が暗殺される。政治的な表立った党派色は薄い

満鉄（南満洲鉄道株式会社）【KR】

  • 大連に本拠を置く準財閥的国策会社
  • 奉天（Fengtian）領内の近代的製造業と大規模農業をほぼ独占
  • 奉天軍内部に**「親日派」vs「親張派」**の対立を生む
  • 日本の大陸権益の経済的基盤

経済状況（1936年時点）【KR】

時期 出来事 影響
1918年 米騒動 経済悪化の兆候
1919年 フランス陥落（最大貿易相手国の喪失） 貿易崩壊・産業不振・大量貧困
1923年 関東大震災 + アナーキスト蜂起 壊滅的破壊
1925年 英国革命 + ドイツの経済覇権確立 さらなる打撃
1927年 震災手形問題 → 台湾銀行破綻 → 銀行恐慌（昭和金融恐慌） 金融システム動揺
1931〜36年 犬養内閣の介入主義的回復政策 緩やかな回復。ただし農村の困窮は継続
1936年 Black Monday（ドイツ発の世界恐慌） 新たな経済危機の波

経済の構造的弱点

  • 資源貧国： 石油・鉄・ゴム等を海外輸入に依存
  • 太平洋補給線の脆弱性： ドイツ東洋艦隊が脅威
  • 農村と都市の格差： 農村困窮が過激主義の温床
  • 貿易先の制約： 旧協商諸国（英仏）の崩壊で市場喪失

8. 対外関係

ドイツ帝国（ライヒスパクト）— 最大の敵対国 【KR】

項目 内容
関係 敵対的。形式上は交戦状態が継続
対立の焦点 太平洋権益、中国での影響力競争
ドイツの東洋艦隊 シンガポール拠点。「帝国海軍最強の海外部隊」
中国問題 ドイツが1928年上海会議で清朝復辟を支援 → 日本の大陸権益と正面衝突
青島協定の不満 日本国民に極めて不人気。都市部で抗議・暴動が発生

ロシア — 緊張関係 【KR】

  • 日本が北樺太・ウラジオストクを占領中 → ロシアの領土回復要求
  • 1928年奉天紛争：日本が奉天を支援しロシアに決定的勝利
  • 沿アムール共和国が日本の対ロ緩衝国

亡命協商（カナダ・豪州）— 複雑な関係 【KR】

  • 旧英仏の亡命勢力とは反独で利害が一致する部分あり
  • ただし：
    • ヴェルサイユで英国が日本要求を拒否した経緯 → 不信感
    • 豪州は太平洋での日本拡大を警戒
    • 「敵の敵は味方」だが完全信頼には程遠い

第三インターナショナル — イデオロギー的対立 【KR】

  • サンディカリスム＝日本にとってのイデオロギー的脅威
  • 1923年にアナーキスト蜂起を鎮圧した経験
  • 片山潜（社会大衆党創設者、サンディカリスト指導者）の暗殺が五・一五事件の引き金
  • 国内左派がインターナショナルとの連帯を模索する動きも

アメリカ — 地域的競合 【KR】

  • 太平洋の競合相手（ただしアメリカ自体が内戦前夜で弱体化）
  • アメリカの参戦不在がカイザーライヒ世界の根幹

奉天政府 — 最も近い同盟国/傀儡 【KR】

  • 日本が唯一承認する中国正統政府
  • 満鉄が経済を支配 → 奉天軍内に親日派 vs 親張派の対立
  • 関係が改善すれば共栄圏加盟、悪化すれば奉天が対日開戦する可能性
  • 奉天戦争で東北経済はほぼ破産

清朝（ドイツ支援）— 敵対的 【KR】

  • ドイツ支援による清朝復辟が日本の大陸権益を脅かす
  • 1928年上海会議で日本は租界都市での評議会議席を確保
