# 用語集

## 世界設定

| 用語 | 説明 |
|------|------|
| カイザーライヒ (KR) | ドイツ帝国が第一次世界大戦に勝利した世界線。本作の前提 |
| Weltkrieg | 第二次世界大戦に相当する戦争（1939年〜）。ドイツ帝国 vs フランス・コミューン |
| フランス・コミューン | フランス革命後のサンディカリスト政権。英連合と同盟 |
| GEA（大東亜共栄圏） | ドイツ帝国の東アジア勢力圏。シンガポールに東洋艦隊を駐留 |
| Black Monday | 1936年、ベルリン証券取引所の崩壊。世界恐慌の引き金 |
| ミッテルオイローパ | ドイツ帝国を中心とした欧州経済圏 |

## 日本の政治勢力

## 連邦

| 用語 | 説明 |
|------|------|
| 日本人民共和国 | 1944年建国。革命後の日本の国号 |
| 連邦評議会 | 連邦の最高意思決定機関。構成国代表による合議制 |
| 構成国 | 連邦を構成する自治体（日本本土、朝鮮、台湾等） |
