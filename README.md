## GitHub Pages

設定資料集と本編をブラウザで閲覧できます。

- **目次**: `pages/index.html`
- **本編**: `pages/ch-*.html`（プロローグ〜終章）
- **設定資料**: `pages/world-setting.html` ほか4ページ

### ページ生成

```bash
npm install
node build-pages.js
```

`manuscript/` 配下の Markdown から `pages/` に HTML を生成します。

## ディレクトリ構成

```
manuscript/     原稿（正本）
pages/          GitHub Pages 用 HTML（自動生成）
docs/           （なし — 設定資料は root の .md）
```
